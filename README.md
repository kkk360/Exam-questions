# 智能出题系统

一款专为教师设计的桌面端智能出题系统，支持多种题型管理、试卷组卷、答案卡生成及导出功能。

## 软件功能

### 题目管理
- **多题型支持**：单选题、多选题、填空题、大题
- **题目属性**：学科、章节、难度等级（基础/简单/中等/较难/困难）
- **LaTeX公式**：支持数学公式编辑和渲染
- **标签系统**：灵活的标签分类管理
- **批量操作**：支持批量删除、导入导出

### 试卷组卷
- **可视化组卷**：拖拽式题目选择和排序
- **分段管理**：支持试卷分段（如选择题、填空题、解答题）
- **页面配置**：支持A4/B5/Letter纸张、页边距、字体大小设置
- **试卷预览**：实时预览试卷排版效果
- **试卷复制**：快速复制已有试卷

### 答案卡管理
- **自动生成**：根据试卷自动生成答案卡
- **答案导出**：支持导出答案卡为PDF格式

### 数据管理
- **本地存储**：所有数据本地存储，保护隐私
- **导入导出**：支持题目和试卷的JSON格式导入导出
- **回收站**：试卷删除后可恢复，防止误删

### 导出功能
- **PDF导出**：高质量PDF试卷导出
- **Word导出**：支持Word格式导出，便于二次编辑

### 系统设置
- **主题切换**：支持亮色/暗色主题
- **数据目录**：自定义数据存储位置
- **偏好设置**：默认学科、难度、常用标签等

## 技术架构

- **前端**：React 18 + TypeScript + Ant Design 6
- **后端**：Electron + Node.js
- **状态管理**：Zustand
- **构建工具**：Electron Vite
- **公式渲染**：KaTeX

由于本软件安装包未购买代码签名证书，Windows 会拦截运行。

## 运行前必须关闭以下两项 Windows 安全功能

### 1. 关闭智能应用控制（Smart App Control）

- 打开 **Windows 安全中心**
- 点击 **应用和浏览器控制**
- 点击 **智能应用控制设置**
- 将开关设为 **关闭**

### 2. 关闭基于声誉的保护

- 打开 **Windows 安全中心**
- 点击 **应用和浏览器控制**
- 点击 **基于声誉的保护设置**
- 关闭以下所有选项：
  - **检查应用和文件**
  - **对 Microsoft Edge 的 SmartScreen**
  - **对 Microsoft Store 应用的 SmartScreen**
  - **网络钓鱼防护**

### 3. 运行安装包

关闭上述两项功能后，双击 `智能出题系统 Setup x.x.x.exe` 正常安装即可。

## 开发指南

### 环境要求
- Node.js >= 18
- npm >= 9

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建打包
```bash
# Windows 64位
npm run build:win

# 仅构建目录（不打包安装程序）
npm run build:unpack
```

### 代码检查
```bash
# ESLint 检查
npm run lint

# TypeScript 类型检查
npm run typecheck

# Prettier 格式检查
npm run format
```

### 代码格式化
```bash
# ESLint 自动修复
npm run lint:fix

# Prettier 自动格式化
npm run format:fix
```
